# 芯片供应链技能库 (Chip Supply Chain Skills)

本项目为OpenClaw类系统提供专业技能支持，涵盖射频工程、元器件认知与选型等多个领域的专业知识和解决方案。

## 技能模块

| 技能名称 | 描述 | 触发场景 |
|---------|------|---------|
| `rf-engineer-expert` | 射频工程领域专业知识和解决方案 | 射频理论、阻抗匹配、传输线理论、5G/WiFi/物联网射频应用 |
| `maxscend-product-support` | 卓胜微(Maxscend)射频产品线技术支持 | 射频开关、LNA、滤波器、射频模组等产品选型、应用设计、调试问题 |
| `rf-circuit-design` | 射频电路设计与调试专业指导 | 射频电路原理图设计、PCB布局、阻抗匹配、调试技巧 |
| `rf-test-instruments` | 射频测试仪器使用指导 | 频谱分析仪、网络分析仪、功率计等仪器操作、测试方法、问题排查 |
| `rf-customer-support` | 射频客户项目技术支持全流程指导 | 射频项目发掘、器件选型、Design-In流程、客诉处理 |
| `component-selection` | 元器件认知与选型专家 | 芯片选型、国产替代方案、元器件参数分析、应用场景匹配等 |
| `hardware-design` | 硬件设计与工艺专家 | PCB Layout、原理图设计、SMT/PCBA制造流程、工艺问题解决等 |
| `tech-support` | 技术支持与调试专家 | 芯片应用调试、电源调试、FPGA/ASIC调试、故障分析与解决等 |
| `quality-control` | 品质与认证把控专家 | 质量标准解读、元器件鉴别、供应链品质把控、合规性认证等 |
| `supply-demand` | 供需匹配与资源整合专家 | 供应链对接、库存资源整合、供应商管理、一站式采购等 |
| `inventory-management` | 库存与现货管理专家 | 库存周转策略、现货储备调整、缺货/涨价应对、市场行情分析等 |
| `price-analysis` | 价格行情分析与定价专家 | 价格跟踪、涨跌分析、定价策略、行情预判等 |
| `cross-border-supply` | 跨境供应链操作专家 | 进出口报关、国际物流、海外仓储、跨境采购与交付等 |
| `pcba-manufacturing` | PCBA柔性制造服务专家 | 中小批量PCBA柔性生产、高频换线、快速出货、BOM配单、SMT贴片等 |
| `custom-solution` | 定制化方案设计专家 | 物联网/消费电子方案设计、芯片选型、元器件整合、整体解决方案等 |
| `engineer-community` | 工程师生态运营专家 | 工程师社区搭建、开发板服务、工具耗材、技术教程、活动策划等 |
| `supply-chain-finance` | 供应链金融与账期服务专家 | 账期服务、融资支持、中小企业服务、数据驱动金融策略等 |
| `customer-service` | 客户需求挖掘与服务专家 | 需求挖掘、终端客户服务、经销商服务、一对一服务、售后服务等 |

**说明**：`agent-browser`、`find-skills` 和 `weather` 是开源技能，本项目中仅为拷贝使用。

## 使用范例

### 范例1：射频电路设计咨询

**用户问题：**
> 我在设计一个GPS接收机的LNA电路，LNA需要2.7V供电，电流约10mA。请问偏置电路应该如何设计？电感选择有什么注意事项？去耦电容应该如何配置？射频输入端的隔直电容应该选择多大容值？需要注意什么参数？

**触发技能：** `rf-circuit-design`

**系统回复：**

针对GPS接收机LNA的偏置电路设计，以下是详细的设计指导：

## 1. 偏置电路设计

### 标准拓扑结构

```
              Vcc (2.7V)
               │
              ┌┴┐
              │ │ Rbias
              └┬┘
               ├──────────────────┐
               │                  │
              ┌┴┐              ┌┴┐
         Lchoke│ │         Cblk│ │ (隔直)
              └┬┘              └┬┘
               │                │
              ┌┴┐              ┌┴┐
         Cdec1│ │              │ │ LNA
              └┬┘              ┤ │
               │                └┬┘
              GND              GND

去耦电容 (金字塔式配置):
Vcc → [10µF/0603] → [100nF/0402] → [10pF/0402] → LNA_Vcc
```

### 偏置电阻计算

```
公式: Rbias = (Vcc - Vds) / Id

您的设计参数:
- Vcc = 2.7V
- Vds = 约2V (典型LNA工作电压)
- Id = 10mA

计算: Rbias = (2.7 - 2.0) / 0.01 = 70Ω

实际选取: 68Ω (最接近标准值)
```

## 2. 电感选择注意事项

### GPS L1频段 (1575.42MHz) 电感选型

| 参数 | 要求 | 说明 |
|-----|------|------|
| **电感值** | 22-47nH | 在1575MHz提供足够阻抗 |
| **自谐振频率(SRF)** | > 3GHz | 至少2倍工作频率 |
| **Q值** | > 30 @ 1GHz | 高Q值减少插入损耗 |
| **额定电流** | > 20mA | 留有50%余量 |
| **封装** | 0402/0201 | 减少寄生参数 |

### 推荐型号

| 厂商 | 型号 | 电感值 | SRF | Q值@1GHz |
|-----|------|--------|-----|----------|
| **Murata** | LQG15HS27NJ02 | 27nH | 4.5GHz | 35 |
| **Murata** | LQG15HS33NJ02 | 33nH | 4.0GHz | 32 |
| **Coilcraft** | 0201HP-27NXGLU | 27nH | 4.2GHz | 45 |
| **TDK** | MLG0603P27NHTD25 | 27nH | 3.5GHz | 38 |

**推荐选择**: Murata LQG15HS27NJ02 (27nH, 4.5GHz SRF)

### 选型检查清单
```
□ SRF > 2 × 1575MHz = 3.15GHz
□ Q值 > 30 @ 1GHz测试频率
□ 额定电流 > 10mA × 1.5 = 15mA
□ 封装为0402或0201
□ 容差 ±5%或更好
```

## 3. 去耦电容配置

### 金字塔式去耦原则

```
         大容量电容          中容量电容         小容量电容
        (滤低频噪声)        (滤中频噪声)       (滤高频噪声)
              │                  │                  │
Vcc ──────────┴──────────────────┴──────────────────┴──── LNA_Vcc
              │                  │                  │
             GND                GND                GND
              ▲                  ▲                  ▲
           最近电源           靠近器件            紧贴引脚
           入口处

推荐配置:
┌──────────────────────────────────────────────────────┐
│  10µF (0603, X5R/X7R)  │  滤除 <1MHz 噪声         │
│  100nF (0402, X7R)      │  滤除 1-100MHz 噪声     │
│  10pF (0402, C0G/NP0)   │  滤除 >100MHz 噪声      │
└──────────────────────────────────────────────────────┘
```

### 布局要求

```
Vcc输入 ──┬──[10µF]──┬──[100nF]──┬──[10pF]──┬── LNA_Vcc
         │          │           │          │
        GND        GND         GND        GND
         │          │           │          │
       电源平面   PCB表层    紧贴器件    Pin脚旁

关键原则:
- 10µF 最靠近电源入口
- 100nF 靠近LNA电源引脚
- 10pF 紧贴LNA电源引脚
- 所有电容接地端直接打过孔到地平面
```

## 4. 隔直电容选择

### 容值计算

```
目标: 在GPS频率(1575MHz)呈现低阻抗

容抗公式: Xc = 1 / (2πfC)

推荐范围: 10-100pF

计算验证:
- 10pF @ 1575MHz: Xc ≈ 10Ω
- 22pF @ 1575MHz: Xc ≈ 4.6Ω  ← 推荐
- 47pF @ 1575MHz: Xc ≈ 2.2Ω
- 100pF @ 1575MHz: Xc ≈ 1Ω

结论: 推荐 22pF 或 47pF，22pF更常用
```

### 关键参数要求

| 参数 | 要求 | 说明 |
|-----|------|------|
| **容值** | 22pF (推荐) | 47pF也可 |
| **耐压** | > 10V | 防止ESD损伤 |
| **Q值** | > 100 @ 1GHz | 高Q减少插入损耗 |
| **封装** | 0402/0201 | 减少寄生电感 |
| **精度** | ±5% (C0G/NP0) | 保证一致性 |
| **温度特性** | C0G/NP0 | 高稳定性，漂移小 |

### 推荐型号

| 厂商 | 型号 | 容值 | 耐压 | Q值@1GHz | 温度特性 |
|-----|------|------|------|----------|----------|
| **Murata** | GRM1555C1H220JA01 | 22pF | 50V | >200 | C0G |
| **TDK** | CGA2B2C0G1H220J050BA | 22pF | 50V | >200 | C0G |
| **AVX** | 04023A220JAT2A | 22pF | 25V | >150 | C0G |

**推荐选择**: Murata GRM1555C1H220JA01 (22pF, 50V, C0G)

## 5. 完整电路设计

### 推荐电路

```
设计参数:
- 频率: 1575.42MHz
- 供电: 2.7V
- 电流: 10mA
- 目标NF: <1dB

完整电路:
                                    ┌─────────┐
VCC(2.7V) ──┬──[10µF, 0603]──┬──[100nF, 0402]──┬──[10pF, 0402]──┬──┤ LNA Vcc
            │                 │                 │                │  └─────────┘
           GND              GND               GND              GND
            │                 │                 │                │
            └─────────────────┴─────────────────┴────────────────┘
                                              │
                                             ┌┴┐
                                             │ │ Rbias = 68Ω
                                             └┬┘
                                              │
                                    ┌─────────┴─────────┐
                                    │                   │
GPS天线 ──┬──[ESD保护]──[匹配网络]──┤                   │
          │                         │                   │
         GND                       ┌┴┐              Cblk = 22pF
                                   Lchoke            ┌┴┐
                                   = 27nH            │ │ LNA
                                   (Murata          ┌┴┐
                                    LQG15HS27NJ02)   │ │ RFin
                                   └┬┘              └┬┘
                                    │                 │
                                   GND               GND
```

### PCB布局要求

```
┌─────────────────────────────────────────────┐
│                                             │
│   GPS天线连接器                              │
│       │                                     │
│      [ESD]  ←─ 靠近连接器                   │
│       │                                     │
│   [输入匹配网络]  ←─ 保持对称，紧凑布局      │
│       │                                     │
│      [LNA]   ←─ 输入端朝上/朝左            │
│       │                                     │
│   [输出匹配/滤波]                            │
│       │                                     │
│   [偏置电路]  ←─ 远离射频输入路径           │
│                                             │
│   去耦电容: 10µF → 100nF → 10pF           │
│   沿电源路径依次放置                         │
│                                             │
│   所有GND过孔: 间距<100mil (2.5mm)         │
│                                             │
└─────────────────────────────────────────────┘

关键要点:
1. LNA输入走线 < 5mm
2. 输入端下方完整地平面
3. 偏置电路与射频输入保持距离
4. 去耦电容紧贴电源引脚
5. 远离数字信号和时钟线
```

### 范例2：射频客户项目支持咨询

**用户问题：**
> 我们已经完成了射频前端的原理图设计，想请卓胜微帮忙审核一下。请问审核流程是怎样的？需要提供什么文件？审核周期大概多久？审核会关注哪些方面？如果发现问题会怎么处理？

**触发技能：** `rf-customer-support`

**系统回复：**

根据卓胜微技术支持流程，以下是原理图审核的详细说明：

#### 1. 提交资料清单

您需要准备以下文件：

| 资料类型 | 具体内容 | 格式要求 |
|---------|---------|---------|
| **原理图** | 完整射频前端原理图（包含开关、LNA、滤波器、PA等） | PDF + 源文件（OrCAD/Altium/PADS） |
| **BOM清单** | 器件清单，包含型号、位号、参数 | Excel格式 |
| **规格书** | 目标频段、功率等级、灵敏度要求等 | Word/PDF |
| **平台信息** | 主芯片型号、参考设计版本 | 文字说明 |
| **特殊要求** | 尺寸限制、成本目标、认证要求 | 文字说明 |

#### 2. 审核流程

```
提交资料 → 分配FAE → 技术审核 → 出具报告 → 问题沟通 → 修改确认
   ↑                                                    ↓
   └────────────── 迭代审核（如需） ←───────────────────┘
```

**各阶段时间：**
- 资料接收确认：1个工作日内
- 技术审核：3-5个工作日
- 报告出具：审核完成后1个工作日
- **总计周期：5-7个工作日**

#### 3. 审核关注要点

**电气设计审核：**
- [ ] **器件型号**：确认选型正确，符合频段要求
- [ ] **引脚连接**：检查电源、地、控制信号连接
- [ ] **电源设计**：供电电压、电流能力、去耦电容配置
- [ ] **匹配网络**：L型/π型/T型拓扑合理性
- [ ] **控制接口**：GPIO/MIPI RFFE连接正确性

**可靠性设计审核：**
- [ ] **ESD保护**：天线端、接口端ESD器件
- [ ] **过压保护**：电源输入保护电路
- [ ] **热设计**：PA等大功率器件散热考虑
- [ ] **EMI抑制**：滤波、屏蔽设计

#### 4. 问题处理方式

| 问题级别 | 定义 | 处理方式 |
|---------|------|---------|
| **严重** | 可能导致功能失效或可靠性问题 | 必须修改，重新审核 |
| **一般** | 影响性能或存在风险 | 建议修改，确认后通过 |
| **提示** | 优化建议 | 可选采纳，记录备案 |
